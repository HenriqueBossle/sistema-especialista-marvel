Sistema Especialista em Personagens da Marvel

Este projeto é um Sistema Especialista desenvolvido em JavaScript com o objetivo de identificar personagens da Marvel com base nas respostas fornecidas pelo usuário. O sistema utiliza um conjunto de regras lógicas definidas a partir do diagrama desenvolvido no Excalidraw:

Diagrama:
https://excalidraw.com/#room=e3b236442761e329df02,ZfQC6wjspoFEAD_BiNAwBw

O sistema funciona fazendo perguntas ao usuário e, a partir das respostas, chega no resultado mais adequado.

Este projeto foi desenvolvido como parte de uma avaliação da disciplina de Fundamentos de Inteligência Artificial.
